# Robô de Automação de Cotações

Este projeto é um script automatizado desenvolvido em Python que coleta as cotações atuais do Dólar e do Euro através de uma API e armazena os dados historicamente em um banco de dados SQL (SQLite).

## Tecnologias Utilizadas
- **Python 3**
- **Biblioteca Requests** (para integração e consumo de API)
- **SQLite / SQL** (para persistência de dados)

## Como executar
1. Instale as dependências: `pip install requests`
2. Execute o script: `python rastreador_moedas.py`
